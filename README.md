# j2team-daily-news
J2TEAM Daily News implementation.

- Based on VnExpress RSS
- Using SvelteKit for both API & UI

#### [news.nomi.dev](https://news.nomi.dev)

### Development

```bash
$ yarn            # install deps
$ yarn dev        # run dev
```

### Build production

```bash
$ yarn build      # build prod
$ yarn preview    # serve prod
```
